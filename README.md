# typescript-basics# TypeScript Basics

Базовий проєкт для вивчення TypeScript з налаштуванням компілятора та прикладами використання основних типів даних.

## Технології
- TypeScript
- Node.js

## Структура проєкту
- `src/` - вихідний TypeScript код
- `dist/` - скомпільований JavaScript код
