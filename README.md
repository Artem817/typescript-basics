# Практична робота №2

У роботі реалізовано:

* **Модальне вікно:** Відкриття та закриття через маніпуляцію CSS-класами (.show) у TypeScript.

* **Fetch API:** Асинхронне отримання списку користувачів з jsonplaceholder, їх типізація через інтерфейс User та динамічний рендеринг у DOM.

* **Scroll Event:** Зміна стилів хедеру (додавання тіні) при прокручуванні сторінки вниз."
